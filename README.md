## ✨ Features
* **Advanced Interaction:** Implements an alternative mechanic for cross-play servers to handle offhand actions natively.
* **Smart Mechanics:** Utilizes precise sneak and click detection to eliminate ghost blocks.
* **Performance-First Code:** Highly optimized listener execution to maintain stable server TPS under heavy load.

## 🎮 Usage & Mechanics (Hướng dẫn sử dụng)

### 🇺🇸 English
* `/geyser offhand` — Swaps the item/block currently held in your main hand into your offhand slot.
* `/oha` — Toggles the specialized offhand interaction mode on or off.
* **Block Placement Mechanic:** Once `/oha` mode is active, **sneak (crouch) and double-click (right-click)** to instantly place the block assigned to your offhand slot.

### 🇻🇳 Tiếng Việt
* `/geyser offhand` — Chuyển đổi nhanh vật phẩm/block từ tay chính sang ô tay phụ (offhand).
* `/oha` — Bật/tắt chế độ tương tác tay phụ đặc biệt.
* **Cơ chế đặt Block:** Sau khi kích hoạt `/oha`, người chơi chỉ cần **ngồi xuống (sneak) và ấn đúp chuột phải (double-click)** để đặt block đang cầm ở tay phụ xuống đất.

## 📥 Installation & Configuration

### Prerequisites / Yêu cầu bắt buộc
* **Java Version:** `Java 21+` (Java 21 or higher required).
* **Minecraft Version:** `1.21.x+` (Minecraft 1.21 or higher required).
* **Server Software:** Spigot, Paper, or Purpur.
* **Dependency:** `Floodgate API` (Required for authentication and system compatibility).

### Setup Steps
1. Download the latest `AllayMC-OffHand.jar` from the GitHub Releases tab.
2. Drop the `.jar` file into your server's `/plugins/` directory.
3. Ensure the latest version of **Floodgate** is also present in the `/plugins/` folder.
4. Restart your server to generate the default configuration files.
5. Verify successful initialization by executing `/plugins` in the console (the plugin name should appear in green).
